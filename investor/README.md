Эта папка содержит инвесторские материалы PaceUp:

- `paceup_deck.pdf` — короткий pitch‑deck (10–12 слайдов).
- `paceup_business_plan.pdf` — полный бизнес‑план (15–25 страниц).
- `paceup_business_plan.html` — HTML версия бизнес‑плана для конвертации в DOCX.

Генерация (Dart/Flutter):

1) `flutter pub get`
2) `dart run tool/investor/generate_pdfs.dart`

Для получения DOCX из HTML:
- Откройте `paceup_business_plan.html` в Microsoft Word.
- Сохраните как `.docx` (Файл → Сохранить как → Формат: Word Document).

Все цифры внутри — черновые допущения, пока не подтверждены фактами.
